Midasr is R package dedicated to estimate restricted MIDAS regression and test whether the restriction conforms to data. The package is developed by Virmantas Kvedaras and Vaidotas Zemlys.

The best way to install package, currently is by using the following code in R:

library(devtools)
install_github("midasr","mpiktas")

Or you can install the particular version using the following code:

library(devtools)
install_github("midasr","mpiktas","v0.0.4")

When the code matures the package will be available from CRAN.
