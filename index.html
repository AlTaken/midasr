Midasr is R package dedicated to estimate restricted MIDAS regression, test whether the restriction conforms to data and use the estimated model for forecasting. The package is developed by Virmantas Kvedaras and Vaidotas Zemlys. The package development was influenced by features of the MIDAS Matlab toolbox created by Eric Ghysels. MIDAS-AR* specification code was written by Julius Vainora.

The best way to install package, currently is by using the following code in R:

library(devtools)
install_github("midasr","mpiktas")

This will install the development version of the package. The latest stable version is 0.0.8 and you can install it using the following code:

library(devtools)
install_github("midasr","mpiktas","v0.0.8")

The package has three demos: okun, RV and logRV. Please run them with the following code: 

library(midasr)
demo("okun","midasr",ask=FALSE)
demo("RV","midasr",ask=FALSE)
demo("logRV","midasr",ask=FALSE)

The last 2 demos take some time to complete (about 5 minutes), so be patient.

Eventually the package will be available from CRAN.

If you find bugs or have suggestions please file an issue. Add the offending R code and the error message for a bug. Or write an email.
